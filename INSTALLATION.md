# Installation Guide

## For Yomi Hustle DEBUG Browser Version

### Step-by-Step Instructions

1. **Download the Mod**
   - Go to the [Releases](../../releases) page
   - Download `gengar-wizard.zip`

2. **Find Your Mods Folder**
   - Navigate to your Yomi Hustle game directory
   - Look for a folder named `mods` (create it if it doesn't exist)
   - Path example: `YOMIHustle/mods/`

3. **Install the Mod**
   - Place the `gengar-wizard.zip` file **directly** into the `mods` folder
   - **Important:** Do NOT unzip the file - leave it as a `.zip`
   - Your mods folder should look like:
     ```
     mods/
     └── gengar-wizard.zip
     ```

4. **Enable the Mod**
   - Start Yomi Hustle
   - If there's a mod manager/loader, enable the Gengar Wizard mod
   - Select the Wizard character to see the Gengar reskin

5. **Verify Installation**
   - The Wizard should now appear as Gengar
   - All animations should work normally
   - If not working, check that the ZIP file is in the correct folder

## Troubleshooting

- **Mod not loading?** Make sure the ZIP file is in the `mods` folder, not unzipped
- **Texture issues?** Clear your game cache and restart
- **Still not working?** Check the game logs for errors related to the mod

## Uninstalling

Simply delete the `gengar-wizard.zip` file from your `mods` folder and restart the game.

# Gengar Wizard Mod for Yomi Hustle

A ZIP client mod that transforms the Wizard character in Yomi Hustle to look like Gengar, the Ghost/Poison-type Pokémon.

## Installation

1. Download the `gengar-wizard.zip` file from the Releases section
2. Place the ZIP file directly into your Yomi Hustle `mods` folder
3. Do NOT unzip it - the game loads ZIP files directly
4. Restart Yomi Hustle
5. The Wizard should now appear as Gengar!

## Folder Structure

```
gengar-wizard.zip
└── wizard/
    ├── character.json
    ├── sprites/
    │   ├── idle.png
    │   ├── attack.png
    │   ├── hurt.png
    │   └── win.png
    └── metadata.json
```

## Features

- Replaces Wizard character sprite with Gengar
- Maintains all original Wizard animations and mechanics
- Compatible with DEBUG browser version

## Notes

- This mod only affects the visual appearance of the Wizard
- All game mechanics and abilities remain unchanged
- Works with Yomi Hustle browser version
